## Part 3 — 10 Common Bugs & Fixes

Document and fix each of these common issues:

### Bug 1: `TemplateDoesNotExist: website/index.html`
**Cause:** Templates not found  
**Fix:** Check `TEMPLATES[0]['DIRS']` in settings.py — must include `BASE_DIR / 'website' / 'templates'`  
**Code fix:** show exact settings.py snippet

### Bug 2: Static files not loading (404 on CSS/JS)
**Cause:** `STATICFILES_DIRS` misconfigured  
**Fix:** Add `STATICFILES_DIRS = [BASE_DIR / 'website' / 'static']` to settings.py  
**Also:** Run `python manage.py collectstatic` if in production mode

### Bug 3: `CSRF verification failed` on POST requests
**Cause:** Missing CSRF token in AJAX calls  
**Fix:** Add `X-CSRFToken` header to all fetch() calls, use `getCsrfToken()` function in main.js

### Bug 4: `No module named 'sklearn'`
**Cause:** scikit-learn not installed  
**Fix:** `pip install scikit-learn numpy --break-system-packages`  
**Also:** Add fallback in recommender.py with try/except ImportError

### Bug 5: `django.db.utils.OperationalError: no such table`
**Cause:** Migrations not run  
**Fix:** `python manage.py makemigrations website && python manage.py migrate`

### Bug 6: `KeyError: 'ANTHROPIC_API_KEY'`
**Cause:** .env file not loaded or key missing  
**Fix:** Check `.env` file exists, add `load_dotenv()` call in settings.py, verify key name matches

### Bug 7: AI Chat returns 500 error
**Cause:** API key invalid or Anthropic API unreachable  
**Fix:** Add try/except around API call in ai_chat view, return fallback message:
```python
except Exception as e:
    return JsonResponse({'reply': 'Sorry, I am temporarily unavailable. Please try again later.', 'status': 'error'})
```

### Bug 8: `UnicodeDecodeError` in seed_data
**Cause:** Special characters in topic names  
**Fix:** Use `# -*- coding: utf-8 -*-` at top of file, ensure all strings are Unicode

### Bug 9: TF-IDF returns no results for valid topics
**Cause:** Similarity threshold too high, or topic corpus empty  
**Fix:** Lower threshold from 0.15 to 0.08, add debug logging to see similarity scores

### Bug 10: Session not persisting bookmarks
**Cause:** Session middleware not enabled or SESSION_ENGINE wrong  
**Fix:** Ensure `django.contrib.sessions.middleware.SessionMiddleware` is in MIDDLEWARE, run `python manage.py migrate` to create session table
